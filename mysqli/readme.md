# SQL Boilerplate :: MySQLi

Welkom bij de MySQLi branch van SQL boilerplate. Hierbij zie je de connectie en uitvoering van de meest voorkomende queries. De manual over MySQLi is hier te vinden: [MySQLi](http://php.net/mysqli). MySQLi is de vervanger van MySQL, de MySQL functies worder deperated in nieuwere PHP versies, dit betekend dat je ze nog steeds kan gebruiken maar dat ze langzamerhand gaan verdwijnen en je dus opzoek moet naar nieuwe functies.

## Download MySQLi branch

Voor het downloaden van de MySQL branch gebruik je de volgende GIT code:

	git clone -b mysqli https://github.com/WouterJ/sql-boilerplate.git

## Hulp bij code

Voer eerst makeTable.php (van de master branch) uit op je localhost. Hierbij worden de juiste tabellen aangemaakt, waardoor je de code kan uittesten. Als je slechts alleen de code wilt bekijken hoef je deze pagina niet uit te voeren.
Het connect.php bestand is het connectie + db select pagina. Deze wordt overal geïncluded.
In alle query bestanden staan queries met de uitwerking daarvan in PHP.
