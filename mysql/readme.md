# SQL Boilerplate :: MySQL

Welkom bij de MySQL branch van SQL boilerplate. Hierbij zie je de connectie en uitvoering van de meest voorkomende queries. De manual over MySQL is hier te vinden: [MySQL](http://php.net/mysql). De homepagina is hier te vinden: [MySQL.com](http://www.mysql.com).

## Download MySQL branch

Voor het downloaden van de MySQL branch gebruik je de volgende GIT code:

	git clone -b mysql https://github.com/WouterJ/sql-boilerplate.git

## Hulp bij code

Voer eerst makeTable.php uit op je localhost. Hierbij wordt de juiste tabel aangemaakt, waardoor de juiste tabellen en databases worden aangemaakt. Als je slechts alleen de code wilt bekijken hoef je deze pagina niet uit te voeren.
Het connect.php bestand is het connectie + db select pagina. Deze wordt overal geïncluded.
In alle query bestanden staan queries met de uitwerking daarvan in PHP.
